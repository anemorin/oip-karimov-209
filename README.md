# Задание 1 — Краулер

## Описание задания

1. Скачать минимум 100 текстовых страниц с помощью краулера из предварительно подготовленного списка.
2. Записать каждую страницу в текстовый файл («выкачка») вместе с HTML-разметкой.
3. Создать файл `index.txt`, в котором хранится ссылка на каждую скачанную страницу.

## Запуск

Требования: **Python 3** (стандартная библиотека, без дополнительных пакетов).

```bash
cd "#1"
python3 crawler.py
```

После выполнения:
- Папка `pages/` будет содержать 100 файлов `001.html` — `100.html`
- Файл `index.txt` будет содержать 100 ссылок

# Задание 2 — Токенизация и лемматизация

## Описание задания

1. Из скачанных HTML-страниц (задание 1) выделить отдельные слова — **токенизация**.
2. Список токенов очищается:
   - без дубликатов
   - без стоп-слов (союзы, предлоги, артикли и т.д.)
   - без чисел и токенов, содержащих цифры
   - без «мусора» (обрывки разметки, смешанные символы и т.д.) — остаются только чисто алфавитные слова длиной ≥ 2
3. Токены **лемматизируются** с учётом части речи (POS-теггер NLTK + WordNetLemmatizer).

## Зависимости

```bash
pip3 install nltk beautifulsoup4
```

NLTK-корпуса загружаются автоматически при первом запуске (или вручную):

```bash
python3 -c "
import ssl; ssl._create_default_https_context = ssl._create_unverified_context
import nltk
nltk.download('punkt_tab')
nltk.download('stopwords')
nltk.download('wordnet')
nltk.download('averaged_perceptron_tagger_eng')
nltk.download('omw-1.4')
"
```

## Запуск

```bash
cd "#2"
python3 tokenizer.py
```

Скрипт читает HTML-файлы из `../#1/pages/`, формирует `tokens.txt` и `lemmas.txt` в папке `#2/`.

# Задание 3 — Инвертированный индекс и булев поиск

## Описание задания

1. По скачанным HTML-страницам (задание 1) построен **инвертированный индекс** — отображение каждого токена на множество документов, в которых он встречается.
2. Реализован **булев поиск** с операторами `AND`, `OR`, `NOT` и поддержкой скобок для сложных запросов.

## Запуск

### 1. Построить индекс

```bash
cd "#3"
python3 index.py
```

```bash
python3 search.py "machine AND learning"
python3 search.py "(network OR internet) AND NOT security"
```
